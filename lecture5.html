Serif fonts are used commonly for physical features, while san serif fonts are typically used for cultural features.

Point Labeling

  • Follow best placement (upper right or left first)
  • Consistency in point –label distances
  • Minimize label line breaks
  • Points and Labels on same side of linear features, like coastlines
  • Coastalfeaturelabelsoverthewater

Line Labeling

  • Do not stretch the label out across the linear feature, repeat it
  • Labelontherightside up
  • Maintain consistent spacing between label and linear feature
  • Usesimplecurves

Polygon Labeling

  • Use spacing and curvature to help communicate the size of the area
  • When using space, use capital letters to improve readability
  • Do not use font size to show area size
  • Stagger horizontal alignments to separate multiple labels
