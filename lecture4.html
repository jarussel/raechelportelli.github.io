Properties of Visual Variables

  • Associative- does not cause the visibility to vary
  • Selective- creates a division between symbols
  • Ordered- represents a visual ordering
  • Quantitative- allow an estimation of numerical difference

Importance to Cartography

Associative variables (size, value) can break the perception of unity.

Selective attributes (ie. hue) should be used when you want to highlight patterns.

Additive Color Space (RGB)

Creates a new color by a process that adds one set of wavelengths to another set of wavelengths.

Subtractive Color Space (CMKY)

Creates a new color by the removal of wavelengths from a light with a broad spectrum of wavelengths.

Lightness

Changes in lightness are percieved in an ordered fashion. Lighter colors are associated with lower values while darker colors are associated with darker values.

Lightness + Hue

Change in lightness can be accentuated by change in hue.
